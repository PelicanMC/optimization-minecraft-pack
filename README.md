## How to install

+ Install qilt 
	+ Download [quilt]([quiltmc.org/](https://maven.quiltmc.org/repository/release/org/quiltmc/quilt-installer/latest/quilt-installer-latest.jar))  
	+ Open quilt-installer  
	+ Click install  
+ Install mod 
	+ Download last releases
	+ Open minecraft folder 
	+ Open mods folder
	+ Extract mods.zip
	+ Open minecraft
